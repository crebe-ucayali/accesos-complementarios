# Accesos Complementarios - AC

Repositorio digital correspondiente al módulo **AC = Accesos Complementarios**, impulsado como **proyecto personal de desarrollo propio** por el **Psicólogo Gabriel Berrospi**.

Este repositorio reúne páginas generales de apoyo, orientación, contacto, directorios, registro de visitas y otros contenidos complementarios que ayudan a organizar la navegación y el acceso a información útil.

## Naturaleza del proyecto

AC forma parte de una iniciativa personal, desarrollada con recursos propios, orientada a responder a necesidades de acceso, organización de información, inclusión, accesibilidad y disponibilidad de herramientas digitales de apoyo.

## Autoría

El desarrollo original de este repositorio corresponde a:

**Psicólogo Gabriel Berrospi**

La autoría comprende la estructura, organización funcional, diseño de navegación, integración de páginas, criterios de presentación, adaptación de contenidos y documentación interna.

## Propósito del repositorio

AC tiene como propósito funcionar como un espacio personal de apoyo transversal. Organiza páginas auxiliares necesarias para orientar, informar, facilitar rutas de acceso, ordenar directorios y reunir recursos complementarios.

## Funciones principales

- Organizar accesos generales del proyecto.
- Facilitar páginas informativas como contacto, presentación, acciones y líneas de trabajo.
- Centralizar directorios ciudadano e informativo.
- Incorporar recursos complementarios de orientación, galería, calendario o registro de participación.
- Mantener una documentación mínima que sustente estructura, autoría, finalidad y criterios de actualización.

## Alcance

Este repositorio está orientado a la consulta, orientación y navegación. Su contenido tiene una finalidad informativa, educativa y de apoyo. No reemplaza servicios oficiales, trámites formales ni canales de atención de entidades externas.
