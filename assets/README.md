# Recursos visuales - AC

Esta carpeta está destinada a los recursos visuales generales utilizados por el repositorio **Accesos Complementarios - AC**, desarrollado como parte del **proyecto personal** del **Psicólogo Gabriel Berrospi**.

## Finalidad

Organizar archivos de apoyo visual como logotipos, íconos, patrones, imágenes u otros elementos gráficos necesarios para la presentación del repositorio cuando estos recursos deban alojarse localmente en AC.

## Propósito funcional

Los recursos visuales ayudan a mantener la coherencia entre páginas y la accesibilidad visual de los contenidos.

## Criterios de uso

- Usar imágenes optimizadas para web.
- Incluir texto alternativo adecuado cuando los recursos se muestren en HTML.
- Evitar imágenes innecesariamente pesadas.
- Mantener coherencia con la identidad visual del proyecto.
- Verificar los derechos de uso y la fuente correspondiente cuando se utilicen recursos externos.
- Evitar duplicar recursos que ya tengan una ubicación oficial dentro del ecosistema.

## Organización vigente

La carpeta `assets/` es la ubicación principal para los recursos visuales generales que se incorporen localmente al repositorio AC. No deben mantenerse copias redundantes dentro de `contenido/assets/`.
